# 需求文档

## 项目目标

仿照「风暴 AI」制作一个专门用于 AI 生图和 AI 视频生成的网站，整体功能和体验可以与参考网站高度相似。

参考网站：

- 风暴 AI：<https://www.fengbaoch.com/app/home>

## 核心功能

网站主要提供两类模型能力：

- 生图模型
- 视频模型

## 生图功能

生图模块需要支持用户输入提示词并选择生成参数。

### 参数选项

- 图片清晰度选择
- 图片比例选择

## 视频生成功能

视频模块需要支持用户输入提示词并选择生成参数。

### 参数选项

- 视频时长选择
- 视频清晰度选择

## 历史项目

网站需要提供客户历史项目功能，方便用户查看、管理或复用过往生成记录。

历史项目至少应包含：

- 生图历史
- 视频生成历史
- 生成状态
- 生成结果预览

## 充值与点数兑换

网站主页需要添加客服微信二维码。

充值流程：

1. 用户在网站主页查看客服微信二维码。
2. 用户添加客服微信。
3. 用户通过客服购买 AI 点数兑换码。
4. 用户在网站内输入兑换码。
5. 系统为用户账户增加对应 AI 点数。

## 模型对接

模型服务直接对接大型上游中转站。

优先考虑：

- API Mart：<https://apimart.ai/zh>

备注：原始需求中提到该中转站已经试用，表现为“很快很稳”。

## 待确认事项

- 是否需要用户登录、注册和账户体系。
- AI 点数的消耗规则，例如不同模型、清晰度、视频时长对应的点数价格。
- 生图比例和清晰度的具体选项。
- 视频时长和清晰度的具体选项。
- 历史项目是否需要支持删除、收藏、重新生成或下载。
- 微信二维码图片素材。
- API Mart 的具体模型、接口文档、鉴权方式和回调机制。
